# Panel Administracyjny - SwooshSpot

## Opis projektu

Panel administracyjny został stworzony jako rozszerzenie projektu sklepu internetowego **SwooshSpot** realizowanego w ramach zajęć na studiach.

Jego głównym zadaniem jest umożliwienie administratorowi prostego zarządzania bazą danych poprzez intuicyjny interfejs. Projekt został napisany zgodnie z architekturą **MVC (Model-View-Controller)** z wykorzystaniem programowania obiektowego w języku PHP.

> **Uwaga:** Projekt nie został jeszcze ukończony w 100%. Część funkcjonalności jest nadal rozwijana i będzie rozszerzana w kolejnych wersjach.

---

# Funkcjonalności

Panel umożliwia między innymi:

* przeglądanie danych zapisanych w bazie,
* dodawanie nowych rekordów,
* edycję istniejących rekordów,
* usuwanie rekordów,
* wyświetlanie podstawowych statystyk na stronie głównej,
* prezentację danych w postaci wykresu,
* wykonywanie własnych zapytań SQL *(wersja BETA)*,
* korzystanie z przykładowych zapytań SQL przygotowanych dla każdej tabeli.

---

# Dashboard

Strona główna panelu administracyjnego zawiera podstawowe informacje dotyczące działania aplikacji.

Aktualnie znajdują się tam między innymi:

* podstawowe statystyki,
* wykres prezentujący dane,
* szybki dostęp do modułów administracyjnych,
* możliwość wykonywania własnych zapytań SQL *(BETA)*.

---

# Zarządzanie bazą danych

Każda tabela dostępna w panelu umożliwia wykonywanie podstawowych operacji CRUD.

Administrator może:

* przeglądać rekordy,
* dodawać nowe dane,
* edytować istniejące wpisy,
* usuwać rekordy.

Dodatkowo dla każdej tabeli przygotowano przykładowe zapytania SQL, które prezentują sposób pracy z danymi zapisanymi w bazie.
